<Section id="el-paquete-io" title="El paquete `io`: Reader, Writer, Closer">

El paquete `io` define las interfaces mĂˇs reutilizadas de la stdlib. Cada una tiene un solo mĂ©todo y se combinan por composiciĂłn.

<Section id="leer-de-cualquier-fuente" title="Leer de cualquier fuente con `io.Reader`">

La belleza de `io.Reader` es que cualquier funciĂłn que recibe uno puede consumir bytes de archivos, redes, strings o memoria sin saber la diferencia.

<CodeBlock
  tabs={[
    {
      lang: "go",
      label: "Go",
      code: `package main

import (
	"fmt"
	"io"
	"strings"
)

// countBytes acepta cualquier io.Reader
func countBytes(r io.Reader) (int, error) {
	buf := make([]byte, 1024)
	total := 0
	for {
		n, err := r.Read(buf)
		total += n
		if err == io.EOF {
			return total, nil
		}
		if err != nil {
			return total, err
		}
	}
}

func main() {
	// strings.NewReader produce un io.Reader desde un string
	r := strings.NewReader("Hola, Gophers!")
	n, _ := countBytes(r)
	fmt.Println("bytes leĂ­dos:", n)
}`
    }
  ]}
  output={"bytes leĂ­dos: 14"}
